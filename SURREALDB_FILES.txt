QUICK START (30 SECONDS)
================================================================================

Step 1: Start SurrealDB
  $ docker run -d --name surrealdb -p 8000:8000 \
    surrealdb/surrealdb:latest start

Step 2: Initialize Database
  $ ./scripts/init-db.sh --seed
  [SUCCESS] SurrealDB is healthy
  [SUCCESS] Successfully imported init.surql
  [SUCCESS] Successfully imported seed.surql

Step 3: Connect & Query
  $ open http://localhost:8000/studio
  Credentials: admin / admin123

  Or via CLI:
  $ surreal sql -e http://localhost:8000 -u admin -p admin123
  > SELECT * FROM agent;

INTEGRATION EXAMPLES
================================================================================

PYTHON:
  from surrealdb import Surreal
  
  async with Surreal() as db:
      await db.connect('http://localhost:8000')
      await db.signin({'user': 'admin', 'pass': 'admin123'})
      agents = await db.select('agent')

REST API:
  curl -X POST http://localhost:8000/sql \
    -u admin:admin123 \
    -H "Content-Type: application/sql" \
    -d "SELECT * FROM agent WHERE status = 'active';"

NODE.JS:
  const { Surreal } = require('surrealdb.js');
  
  const db = new Surreal();
  await db.connect('http://localhost:8000');
  await db.signin({user: 'admin', pass: 'admin123'});
  const agents = await db.select('agent');
